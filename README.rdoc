= cocoapods-kocomojo-sdk

KocomojoSDK utility 

Use it to finalize installation of KocomojoSDK in your XCode project.

To install:

  gem install cocoapods-kocomojo-sdk
  
Usage:

  pod kocomojo-sdk [MyProject.xcodeproj]

This adds a final Build Phase to your project that strips KocomojoSDK of unnecssary architectures.
